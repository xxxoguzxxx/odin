# Frameworks and Preprocessors Bölümü - The Odin Project

Bu bölümde **CSS frameworkleri** ve **ön işlemciler (preprocessors)** hakkında temel bilgiler edindim.  
Bootstrap ve Tailwind gibi frameworklerin geliştirme sürecini hızlandırdığını,  
Sass ve LESS gibi ön işlemcilerin ise değişkenler ve mixin'lerle CSS'i daha modüler hale getirdiğini öğrendim.  
Artık projelerimde daha düzenli ve ölçeklenebilir stiller yazabiliyorum.
