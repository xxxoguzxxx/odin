# Custom Properties Bölümü - The Odin Project

Bu bölümde **CSS değişkenlerini (custom properties)** öğrendim.  
`:root` seçicisiyle global değişkenler tanımlamayı ve `var()` fonksiyonuyla bunları kullanmayı öğrendim.  
Artık projelerimde renk, yazı boyutu gibi değerleri tek bir yerden kontrol edebiliyorum.
