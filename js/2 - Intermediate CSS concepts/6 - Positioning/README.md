# Positioning Bölümü - The Odin Project

Bu bölümde HTML öğelerinin sayfa üzerindeki yerleşimini kontrol etmek için kullanılan **CSS position** özelliklerini öğrendim.  
`static`, `relative`, `absolute`, `fixed` ve `sticky` değerlerinin nasıl davrandığını ve birbirleriyle farklarını kavradım.  
Artık öğeleri tam istediğim yere yerleştirip düzenli bir sayfa yapısı oluşturabiliyorum.
