# More Text Styles Bölümü - The Odin Project

Bu bölümde metinleri biçimlendirmek için kullanılan ek **CSS özelliklerini** öğrendim.  
`text-transform`, `text-decoration`, `letter-spacing`, `line-height` ve `text-align` gibi özelliklerle metinlerin görünümünü nasıl değiştireceğimi kavradım.  
Artık yazıların hem okunabilir hem de estetik görünmesini sağlayabiliyorum.
