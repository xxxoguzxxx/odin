# CSS Functions Bölümü - The Odin Project

Bu bölümde CSS içinde kullanılan **fonksiyonların** nasıl çalıştığını öğrendim.  
`calc()`, `min()`, `max()`, `clamp()` gibi fonksiyonlarla değerleri dinamik şekilde hesaplayabildiğimi gördüm.  
Artık stilleri daha esnek ve uyumlu hale getirmek için bu fonksiyonları kullanabiliyorum.
