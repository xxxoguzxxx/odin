# Creating a Grid Bölümü - The Odin Project

Bu bölümde **CSS Grid** kullanarak nasıl ızgara düzenleri oluşturulacağını öğrendim.  
`grid-template-rows`, `grid-template-columns` ve `gap` özellikleriyle satır ve sütun yapılarını tanımlamayı kavradım.  
Artık sayfa elemanlarını düzenli ve kontrollü bir şekilde yerleştirebiliyorum.
