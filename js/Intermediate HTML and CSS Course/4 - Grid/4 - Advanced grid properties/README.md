# Advanced Grid Properties Bölümü - The Odin Project

Bu bölümde **CSS Grid** sisteminin ileri düzey özelliklerini öğrendim.  
`grid-template-areas`, `minmax()`, `auto-fit`, `auto-fill` ve `fr` birimi gibi kavramlarla daha esnek ve dinamik düzenler oluşturmayı kavradım.  
Artık karmaşık sayfa yerleşimlerini daha az kodla ve daha düzenli bir şekilde tasarlayabiliyorum.
