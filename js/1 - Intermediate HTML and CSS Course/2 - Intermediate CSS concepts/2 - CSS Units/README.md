# CSS Units Bölümü - The Odin Project

Bu bölümde **CSS birimlerini** ve aralarındaki farkları öğrendim.  
Mutlak birimler (`px`, `cm`, `in`) ile göreceli birimler (`em`, `rem`, `%`, `vh`, `vw`) arasındaki farkları kavradım.  
Artık responsive tasarımlar yaparken hangi birimi ne zaman kullanmam gerektiğini biliyorum.
