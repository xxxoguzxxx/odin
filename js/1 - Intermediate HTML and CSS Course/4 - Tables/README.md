# Tables Bölümü - The Odin Project

Bu bölümde **HTML tablolarının** yapısını ve kullanımını öğrendim.  
`<table>`, `<tr>`, `<th>`, `<td>` etiketleriyle tablo oluşturmayı ve tabloya başlık eklemeyi kavradım.  
Ayrıca CSS ile tablo stillerini düzenlemeyi ve verileri okunabilir hale getirmeyi öğrendim.
