# Default Styles Bölümü - The Odin Project

Bu bölümde tarayıcıların HTML etiketlerine varsayılan olarak uyguladığı **stillerin (default styles)** ne olduğunu öğrendim.  
Farklı tarayıcıların farklı varsayılan stiller kullandığını ve bunları sıfırlamak için **CSS reset** veya **normalize.css** gibi yöntemlerin kullanılabileceğini gördüm.  
Artık projelerimde tutarlı bir görünüm için varsayılan stilleri nasıl kontrol edeceğimi biliyorum.
