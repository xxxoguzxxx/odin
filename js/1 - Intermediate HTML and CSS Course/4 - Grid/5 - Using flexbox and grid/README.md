# Using Flexbox and Grid Bölümü - The Odin Project

Bu bölümde **Flexbox** ve **CSS Grid** sistemlerini bir arada kullanmayı öğrendim.  
Hangi durumlarda Grid’in, hangi durumlarda Flexbox’ın daha uygun olduğunu kavradım.  
Artık sayfa düzenlerinde iki yöntemi birlikte kullanarak hem esnek hem de düzenli tasarımlar oluşturabiliyorum.
