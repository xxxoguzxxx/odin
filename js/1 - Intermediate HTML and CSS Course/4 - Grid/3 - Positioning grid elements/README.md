# Positioning Grid Elements Bölümü - The Odin Project

Bu bölümde **CSS Grid** içinde öğelerin konumlarını nasıl kontrol edebileceğimi öğrendim.  
`grid-row`, `grid-column`, `justify-self`, `align-self` ve `place-self` özellikleriyle öğeleri istediğim hücreye yerleştirmeyi kavradım.  
Artık grid düzeninde öğeleri tam olarak istediğim konuma yerleştirebiliyorum.
