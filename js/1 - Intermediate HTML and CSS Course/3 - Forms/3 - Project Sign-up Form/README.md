# Sign Up Form

Bu proje, **HTML ve CSS** kullanılarak hazırlanmış basit ve modern bir kayıt formudur.  
Form, kullanıcıdan ad, e-posta ve şifre bilgilerini alır. Şık bir tasarıma ve responsive (mobil uyumlu) bir yapıya sahiptir.

## Özellikler
- Tamamen HTML ve CSS ile oluşturuldu  
- Duyarlı (responsive) tasarım  
- Gölgelendirme ve renk geçişi ile modern görünüm  
- Kullanıcı dostu form düzeni

## Öğrendiklerim
- Form yapısını oluşturmak ve stillendirmek  
- `input` ve `label` etiketlerini doğru biçimde ilişkilendirmek  
- CSS ile hizalama, renk geçişi (gradient) ve odak efektleri kullanmak
